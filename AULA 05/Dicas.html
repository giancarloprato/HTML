Code snippets

Code Snippets, que numa tradução livre seria algo como “retalhos de código”, são pedaços de código pré-fabricados e parametrizáveis
que podemos inserir em nossa aplicação. É um recurso muito interessante: ajuda na produtividade e na padronização do código.

Exemplos de code Snippets

  • <link rel="stylesheet" href=".css">
    • Esse code é utilizado para conectarmos o .HTML ao .CSS.

  • <footer>
    • A tag footer é utilizada para criarmos o rodapé da página.

  • <meta name="viewport" content="width=device-width,initial-scale=1>
    <http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="HandheldFriendly" content="true">
    • Esse grupo de tags define a proporção de exibição da página no navegador, permitindo em celulares o aumento de zoom.
