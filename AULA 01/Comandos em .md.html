Titulação

Para criar títulos é utlizado a sintaxe de <#> que varia de <#> á <######>
Exemplo : (# h1-h6 titulo)

Ênfase Negrito e Itálico

Negrito a sintaxe utilizada são dois ** ou __ no inicío e fim da palavra, exemplo < ** Negrito ** > ou < __ Negrito __ >
Exemplo : Hoje estou feliz
Itálico a sintaxe utilizada é um * ou _ no começo e no final da palavra ou frase < * Itálico *> ou <_ Itálico _>
Exemplo : Hoje estou feliz

Links

Link direto È a forma de link onde o link é visível, e é utilizado usando a sintaxe [< https://exemplo.com/ >]
Exemplo : https://tech4me.wakke.co/class/student/courses/answer-activities/0r2F25dAdhrLkMEbkmw0OQ/XrHxstit4JdbfQ64ZxgJQA
Texto-Âncora O texto ancora é uma forma de usar o link dentro de uma palavra chave, utilizando a sintaxe < Sintaxe >
Exemplo : Wakke

Lista de itens

Listas não ordenadas (*) no começo da palavra

  • Exemplo

Listas ordenadas: Número seguido de (.)

  1. Exemplo
  2. Exemplo

Imagens

A sintaxe para inserir uma imagem é ![Alt ou título da imagem](URL da imagem)

Citação (Quote)

È feita usando a sintaxe (>) no começo da palavra ou frase, para fazer a citação.

“Só sei que nada sei”

Código (Code Highlight)

Código em linha: é utilizado o (`) no começo e no final do código

Múltiplas Linhas de Código : Para tornar toda uma linha em código é utilizada a sintaxe <`` `> no começo e no final da linha de código.

Tabela

Para criarmos uma tabela utilizamos as seguintes sintaxes <|> (Colunas) e <-> para definir a quebra de linha da tabela.
Exemplo:

Funcionário horas trabalhadas
João 40
Maria 50
Marta 58
Jorge 36

Para definirmos o alinhamento da tabela utilizamos <:> do lado do campo horizontal de hífens <—>

Exemplo :
Alinhado a esquerda: usar : no lado esquerdo (alinhamento padrão);
Alinhado a direita: usar : no lado direito;
Centralizado: usar : dos dois lados.

Como ficará na tabela :

Alinhado a esquerda Centralizado Alinhado a direita
Valor Valor Valor
